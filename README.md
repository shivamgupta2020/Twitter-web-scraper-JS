# Twitter-web-scraper-JS

This project list down top 5 trending trends of Twitter.

# Clone Repo
```
git clone git@github.com:shivamgupta2020/Twitter-web-scraper-JS.git
cd Twitter-web-scrapper-JS
```
# Install Libraries
```
npm install
```
# Start app
```
node app.js
```
# Enjoy app 
```
http://localhost:10000
```
